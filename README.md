### 정리  
- 트리 구조와 트리에서 가장 많이 사용하는 순회 알고리즘  
- 자식 노드가 2개 이하인 노드를 이진 트리  
- 가장 많이 사용되고 있는 개념  
- 전위, 중위, 후위, 단계 순회 알고리즘 4가지 존재  
- 재귀, 스택, 큐를 사용해 구현 가능  
- 재귀호출을 이용하여 쉽게 구현  
- 트리와 스택은 고려해야할 부분이 존재한다.  
- 알고리즘의 특징에 맞춰 재귀 호출, 스택, 큐 중에서 더 잘 맞는 구현 방식을 선택
